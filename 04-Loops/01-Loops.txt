## Sum of All Even Numbers 
Problem statement
Question: Given a number N, print sum of all even numbers from 1 to N.

Answer one: Using Range
N = int(input())

even_sum = 0

# Loop through numbers from 1 to N
for i in range(1, N + 1):
    if i % 2 == 0:
        even_sum += i

# Print the sum of even numbers
print(even_sum)

Answer two: Using while Loop
N = int(input())

# Initialize variables
i = 1
even_sum = 0

while i <= N:
     if i % 2 == 0:
        even_sum += i
     i += 1

print(even_sum)

Problem statement
Question: Write a program to generate the reverse of a given number N. Print the corresponding reverse number.

Note : If a number has trailing zeros, then its reverse will not include them. For e.g., reverse of 10400 will 
be 401 instead of 00401.

Answer: 
n = int(input())
reverse = 0
while n > 0:
    rem = n % 10
    reverse = reverse * 10 + rem  # Build the reversed number
    n //= 10
print(reverse)


## Fibonacci Series 
Question: Print the series in Fibonacci i.e take the input from user to print how many fib. numbers

Answer:  
n = int(input())

second_last_num = 0
last_num = 1

print(second_last_num)
print(last_num)

count = 2 # how many fib numbers we want i.e n

while count <= n:
    next_fib = last_num + second_last_num
    print(next_fib)
    second_last_num = last_num
    last_num = next_fib

    count += 1

## Fibonacci Series 
Question: Print the Last Number(nth) in series for Fibonacci i.e take the input from user to print last number

second_last_num = 0

if n == 1:

#Decimal to Binary
Question: Convert a given (n) number into Binary

Answer: 
n = int(input())
ans = ""

while n > 0:
    if n % 2 == 0:
        #Even
        ans = "0" + ans
    else:
        #Odd
        ans = "1" + ans
    n //= 2
print(ans)

##Binary to decimal
Question: Convert the give binary to decimal

Ans:
n = int(input())

ans = 0 
power_of_2 = 1 # 2 raise to the power 0
while n > 0:
    last_digit = n % 10 # remainder
    ans += last_digit * power_of_2

    power_of_2 *= 2
    n //= 10 #eliminate the last digit
    
print(ans)
